# Service Tizon Scan

Este servicio FastAPI tiene dos guías separadas de levantamiento:

- `README.DOCKER.md`: instalación de Docker + ejecución del proyecto con Docker.
- `README.LOCAL.md`: instalación de Python + PostgreSQL + ejecución local sin Docker.

## ¿Qué guía usar?

- Si quieres una instalación más rápida y aislada, usa `README.DOCKER.md`.
- Si prefieres desarrollo local con entorno virtual, usa `README.LOCAL.md`.
